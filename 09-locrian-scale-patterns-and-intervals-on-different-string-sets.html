09 - Locrian scale patterns and intervals on different string sets

Locrian mode

The locrian mode is often described as dark-sounding one.

This scale has the following structure:

semitones (1 semitone is 1 fret on the fret board)
1-2-2-2-1-2-2-2

Let's take a specific scale, C locrian as an application:

C locrian mode:
1-2--2--2-1-2--2--2
c-db-eb-f-gb-ab-bb-c

Here's where you can find all the notes of C mixolydian or C dominant on each string:
