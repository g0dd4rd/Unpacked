04 - Mixolydian scale patterns and intervals on different string sets

Mixolydian mode - Dominant scale

The mixolydian mode can be used to outline a dominant chord that can be found on a fifth degree of ionian mode / natural major scale. It's also often used in blues or rock harmony, creating their typical chord progressions. This mode has only one flattened degree - the seventh degree, that gives it together with the major third the mixed major / minor tonality. It probably makes it easier to learn because of only one flat in the scale, as well to fret some of the chord voicings.

This scale has the following structure:

semitones (1 semitone is 1 fret on the fret board)
2-2-1-2-2-1-2

Let's take a specific scale, C mixolydian mode - dominant scale as an application:

C dominant scale or / C mixolydiann mode:
2-2-12-2-1-2
c-d-ef-g-a-bb-c

Here's where you can find all the notes of C mixolydian or C dominant on each string:
